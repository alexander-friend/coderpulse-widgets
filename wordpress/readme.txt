A collection of open-source widgets and components from CoderPulse.io.

== Description ==

CoderPulse Widgets brings lightweight, standard-compliant Web Components to your WordPress site. The library currently includes:
* **Cron Builder**: A visual cron expression generator and explainer.
* **Hello World**: A simple greetings widget.
* **Star Rating**: An interactive star rating component.

All widgets include a subtle backlink to CoderPulse.io to help grow the community.

== Installation ==

1. Upload the `coderpulse-widgets` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the block editor (Gutenberg) to add "CoderPulse" blocks, or use shortcodes.

== Shortcodes ==

* `[cp-cron-builder]`
* `[cp-hello-world name="Alex"]`
* `[cp-star-rating rating="4" max="5"]`
